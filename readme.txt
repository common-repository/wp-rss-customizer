You want to custom rss feed or create API from your site. WP Rss Customizer will help you. Support RSS 2.0, RSS 1.0, ATOM, RDF, JSON

== Description ==

Create unlimited feed links. You can custom data in feed. Support Post, Page, WooCommerce, BBPress, Bubby Press etc... Output: RSS 2.0, RSS 1.0, ATOM, JSON, RDF.

### Features

* Support RSS 2.0, RSS 1.0, JSON, ATOM, RDF

== Installation ==

1. Unzip the download package
1. Upload `woo-multi-currency` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
